SMART BABY CARADLE SYSTEM
This system helps the parents to monitor their infant from anywhere of the world using internet of things. Through they can monitor their infant and control temperature around the baby, play music and even interact with them using video calling system.
Technology: Blynk, Arduino IDE
Language: Embedded C
Duration: 23rd June, 2023
